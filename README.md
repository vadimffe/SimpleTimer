# SimpleTimer
Simple timer where you can set your time to execute "ring a bell" command. 

* MVVM
* System.Threading.Timer
* IValueConverter
* .NET 6
* log4net ready
* WPF Material Design (TimePicker)
* Screen Lock/Unlock SessionEvent listener
* WPF
